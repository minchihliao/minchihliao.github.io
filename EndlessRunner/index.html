Project Overview

In this project, I single-handedly developed an endless runner game using Unreal Engine 5, leveraging the powerful Blueprint visual scripting system. The game challenges players with a fast-paced, obstacle-laden environment where they must use quick reflexes to dodge obstacles and collect items for score enhancement.

Objective

To demonstrate proficiency in Unreal Engine and Blueprint scripting through the development of an endless runner game.

Key Features

  • Game Mechanics
      Implemented using Unreal 5 to enable dynamic lateral movements, jumps, and turns allowing players to navigate through obstacles and strive for high scores.
  • Road Generation System
      Developed a customizable road generation tool within Unreal, empowering level designers to easily create varied path designs tailored to gameplay needs, including complex bends and T-junctions.
